# Contributing to quickdb-studio

Thanks for taking the time to contribute!

## Development

### Prerequisites
- Go 1.22+
- Node.js 18+ (recommended 20+)

### Run backend
```bash
export QUICKDB_STUDIO_TOKEN=devtoken
go run ./cmd/quickdb-studio
```

### Run webui (dev server)
```bash
cd webui
npm install
npm run dev
```

## Pull Requests

- Keep changes focused and easy to review
- Prefer small PRs over large refactors
- Add/update documentation when user-facing behavior changes
